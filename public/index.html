Url Shortener Service

  1. Enviar una solicitud POST con la url a [project_url]/api/shorturl/new y recibe un JSON con la url acortada. Ejemplo {"original_url": "www.google.com", "short_url":1}
  2. Si se envia una url que no sigue el formato http(s)://www.example.com(/more/routes) devuelve un JSON con un error. Ejemplo {"error":"invalid URL"}
